# Aprendizagem Baseada em Problemas (PBL) 🤔

- Lógica de programação é como aprender um novo idioma, um que permite problemas em soluções claras
, e ordenadas atraves de passos lógicos e algoritmos 

- Já o pensamento computacional é como ter um mapa que te ajuda a encontrar caminhos
para resolver desafios, projetar sistemas e compreender comportamentos

- Algoritmos é como uma receita de bolo

### Temos os ingredientes!
1. Açucar 
2. Ovos
3. Massa
4. trigo

- resultado: Um delicioso bolo!

# O que compôe um algoritmo? 🧩

- Entrada: Entrada de dados do programa
- Processamento: Onde processaremos e codamos a solução
- Saída: Resultado dos processos anteriores!
